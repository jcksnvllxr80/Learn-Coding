# Learn Coding

A gentle introduction to programming concepts designed for beginners. Each lesson takes just 5 minutes and builds foundational understanding through hands-on activities.

## Core Concepts Covered

**Programming Fundamentals**
- Commands and sequencing
- Loops and repetition
- Logic and conditionals
- Variables and data storage
- Functions and code reuse

**Problem Solving**
- Debugging and error fixing
- Input/output handling
- Data types and manipulation
- Random number generation
- Algorithms and sorting

**Practical Applications**
- Text manipulation and games
- File operations
- Time and date handling
- Simple web concepts
- Creative projects (Mad Libs, games)

## Approach

- **5 minutes per day** - bite-sized learning sessions
- **Hands-on activities** - learn by doing, not just reading  
- **No jargon** - concepts explained in simple terms
- **Progressive building** - each lesson builds on previous knowledge

Uses Python with browser-based tools like trinket.io or replit.com - no installation required.
