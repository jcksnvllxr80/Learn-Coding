**Step 1 (1 min): Set the idea**  

Sometimes you want the computer to do the same thing many times. Instead of writing it over and over, you can make it repeat â€” that's called a _loop_.

---

**Step 2 (2 min): The long way first**  
Type this:

```python
print("Clap!")
print("Clap!")
print("Clap!")
print("Clap!")

```

Run it.  
What if you wanted 100 claps? That would take forever to type!

---

**Step 3 (1 min): The loop way**  
Erase that and type:

```python
for i in range(5):
    print("Clap!")
```

Run it again.  
`for i in range(5):` means 'do this 5 times.' The line under it is what repeats.
